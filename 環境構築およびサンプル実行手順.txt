1. Node.jsのインストール

　(1) 以下のURLより、ご利用のOS用のインストール・パッケージをダウンロードしてください。

　　　　https://nodejs.org/download/

　(2) ダウンロードしたパッケージを実行して、すべてデフォルトの指定でインストールしてください。

2. テスト用ツールのインストール

　(1) コマンド・プロンプトを起動し、以下のコマンドを実行してください。

　　　　npm install -g bower karma protractor
　　　　webdriver-manager update

3. サンプル・アプリケーションの実行

　(1) コマンド・プロンプトでサンプル・アプリケーションのフォルダーの下にカレント・ディレクトリを指定し、
　　　以下のコマンドを実行してください。

　　　　npm install
　　　　bower install

　(2) サンプル・アプリケーションのフォルダーの下に、serverという名前フォルダーがある場合は
　　　コマンド・プロンプトでサンプル・アプリケーションのフォルダーの下にカレント・ディレクトリを指定し、
　　　以下のコマンドを実行してください。サーバー・アプリケーションが起動します。

　　　　npm start

　　　なお、サーバー・アプリケーションを停止したい場合には、実行中のコマンド・プロンプトでCTRL+Cを押します。

4. テストの実行

　(1) Protractorが依存しているツール（webdriver-manager）はJavaの実行環境が必要です。
　　　Oracle社のサイトからJREまたはJDKを入手し、インストールしてください。

　(2) Protractorによるテストを実施する場合は、事前に以下のコマンド・プロンプトにてコマンドを実行します。

　　　　webdriver-manager start

　(3) Jasmine + Karmaのテストを実行する場合には、新しいコマンド・プロンプトを開き、以下のコマンドを実行します。

　　　　npm test

　(4) Jasmine + Protractorのテストを実行する場合には、新しいコマンド・プロンプトを開き、以下のコマンドを実行します。

　　　　protractor e2e-tests/protractor.conf.js
